# AlunoServiceRest
### Tecnologias Usadas:
- JSON
- Crud
- MySQL
- Serializable
- PostMan

- Projeto para adicionar alunos e gerar automaticamente a média e a situação podendo ser consumido pelo Service. Utilizei o POSTMAN como exemplo para consumir o aplicativo.
